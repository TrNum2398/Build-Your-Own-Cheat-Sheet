HTML Cheat sheet

Main-root

<html>
The HTML Document / Root element.

<html lang="en-US">
Set the document language.

Sectioning-root

<body>
The Document Body element.There can be only one <body> element in a document.

Content sectioning

<article>
Represent a self-contained composition in a document, page, application, or site.

<aside>
Asides are frequently presented as sidebars or call-out boxes.

<footer>
Typically contains information about the author of the section, copyright data or links to related documents.

<header>
Represent introductory content, typically a group of introductory or navigational aids.

<h1> to <h6>
Represent six levels of section headings.<h1> is the highest section level and <h6> is the lowest.

<main>
Represent the dominant content of the <body> of a document.

<nav>
The navigation sections are menus, tables of contents, and indexes.

<section>
Represent a generic standalone section of a document.

Text content

<div>
The generic container for flow content. It has no effect on the content or layout until styled in some way using CSS.

<hr>
Horizontal Rule. Represent a thematic break between paragraph-level elements.

<li>
Represent an item in a list. It must be contained in a parent element: an ordered list (<ol>), an unordered list (<ul>)

<ol>
Represent an ordered list of items — typically rendered as a numbered list.

<p>
Represents a paragraph.

<ul>
Represents an unordered list of items, typically rendered as a bulleted list.

Inline text semantics

<a>
Create a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

<a href="https://www.mozilla.com">Mozilla</a>
Linking to an absolute URL.

<a href="./p">Directory-relative URL</a>
Linking to relative URLs.

<p><a href="#Section_further_down"></a>Jump to the heading below</a></p>
Linking to an element on the same page.

<a href="mailto:nowhere@mozilla.org"></a>Send email to nowhere</a>
Linking to an email address.

<b>
Used to draw the reader's attention to the element's contents, which are not otherwise granted special importance. This was formerly known as the Boldface element.

<br>
Produces a line break in text (carriage-return). It is useful for writing a poem or an address, where the division of lines is significant.

<i>
To display text in italic type.

<span>
It can be used to group elements for styling purposes.

<strong>
Browser typically render the contents in bold type.

Image and multimedia

<img>
Embed an image into the document.

<img src="favicon144.png" alt="MDN logo">
Embeds an image into the page and includes alternative text for accessibility.

Table content

<caption>
Specify the caption (or title) of a table.

<col>
define a column within a table and is used for defining common semantics on all common cells.

<colgroup>
Define a group of columns within a table.

<table>
represent tabular data — that is, information presented in a two-dimensional table comprised of rows and columns of cells containing data.

<tbody>
a set of table rows (<tr> elements), indicating that they comprise the body of the table (<table>).

<td>
Define a cell of a table that contains data.

<tfoot>
Define a set of rows summarizing the columns of the table.

<th>
Define a cell as header of a group of table cells.

<thead>
Define a set of rows defining the head of the columns of the table.

<tr>
Define a row of cells in a table.

Forms

<button>
Represent a clickable button, used to submit forms or anywhere in a document for accessible, standard button functionality.

<datalist>
A set of <option> elements that represent the permissible or recommended.

<fieldset>
group several controls as well as labels (<label>) within a web form.

<form>
Represent a document section containing interactive controls for submitting information.

<input>
To create interactive controls for web-based forms in order to accept data from the user.

<label>
Represent a caption for an item in a user interface.

<option>
Define an item contained in a <select>, an <optgroup>, or a <datalist> element.

<select>
Represent a control that provides a menu of options.

<textarea>
A multi-line plain-text editing control, useful when you want to allow users to enter a sizeable amount of free-form text.
